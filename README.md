## 🧠 Sobre o projeto

Este projeto foi desenvolvido durante um **bootcamp realizado na plataforma DIO**, com foco no uso de **inteligência artificial aplicada ao desenvolvimento de software**, especialmente com o GitHub Copilot.

A aplicação foi construída acompanhando as aulas, desafios e metodologias propostas no bootcamp, que priorizam o aprendizado prático por meio da criação de projetos reais. Durante o programa, são abordados conceitos como geração de código com IA, automação de tarefas e melhoria da produtividade no desenvolvimento.

O projeto utiliza tecnologias fundamentais da web, como:
- **HTML**
- **CSS**
- **JavaScript**

Além disso, durante o desenvolvimento foram aplicados conceitos como:
- uso do Git e GitHub  
- apoio de IA (GitHub Copilot) na escrita de código  
- lógica de programação  
- manipulação do DOM  
- boas práticas de desenvolvimento  

## 🎯 Objetivo

O principal objetivo deste projeto foi **aplicar na prática os conhecimentos adquiridos no bootcamp da DIO**, utilizando o GitHub Copilot como ferramenta de apoio para acelerar o desenvolvimento e melhorar a qualidade do código.

O bootcamp tem como proposta ensinar como integrar a IA ao fluxo de desenvolvimento, desde a escrita até testes e documentação, aumentando a produtividade do desenvolvedor.

## 🚀 Aprendizados

Durante a construção do projeto, foi possível:
- entender como utilizar IA no desenvolvimento  
- melhorar a produtividade na escrita de código  
- praticar engenharia de prompts  
- automatizar partes do desenvolvimento  

## 💡 Considerações

Este projeto representa minha evolução durante o bootcamp, demonstrando minha capacidade de utilizar ferramentas modernas como o GitHub Copilot e aplicar, na prática, as metodologias ensinadas pela DIO, que são baseadas em aprendizado ativo, desafios e construção de projetos reais.
